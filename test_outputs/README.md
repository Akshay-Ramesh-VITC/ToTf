# Test Output Files

This directory contains output files generated by the ModelView test suite.

## Multi-Branch Architecture Tests

These visualizations demonstrate ModelView's ability to handle complex neural network topologies:

### Basic Architectures
- **parallel_branches.png** - Simple parallel processing paths that merge
  - 2 parallel branches, each with 2 Dense layers
  - Tests basic parallel visualization

### Inception-Style
- **inception_module.png** - Multi-scale feature extraction (GoogLeNet-style)
  - 4 parallel convolution branches (1x1, 3x3, 5x5, pooling)
  - Tests multi-scale processing visualization

### Complex Connectivity
- **cross_connections.png** - Inter-branch communication
  - Parallel branches with cross-connections
  - Tests non-sequential data flow

- **multi_path_merge_split.png** - Multiple merge and split points
  - Complex routing with multiple cycles
  - Tests handling of merge/split patterns

### Dense Networks
- **densenet_block.png** - DenseNet-style dense connections
  - Each layer connects to all previous layers
  - Tests all-to-all connectivity visualization

### Multi-Input/Output
- **multi_input_multi_branch.png** - Multi-input fusion architecture
  - 3 inputs with complex branching
  - Tests multiple input handling

- **multi_output_multi_branch.png** - Multi-head prediction architecture
  - Shared trunk with 3 output heads
  - Tests multi-task learning visualization

### Residual Networks
- **multiple_skips.png** - Multiple residual skip connections
  - Local and global skip paths
  - Tests multi-level residual visualization

### Maximum Complexity
- **complex_dag.png** - Complex directed acyclic graph (300 DPI)
  - Maximum cross-connectivity without cycles
  - Stress test for complex topologies

## Other Test Outputs

- **verbose_labels.png** - Verbose layer labels with names
- **clean_model.png** - Basic model visualization

## Regenerating Outputs

To regenerate all test outputs:

```bash
# Run all tests (generates all outputs)
pytest test/test_modelview_tf.py -v

# Run only multi-branch tests
pytest test/test_modelview_tf.py::TestMultipleBranchesAndCrossConnections -v
```

All outputs are publication-quality PNG files suitable for research papers.
